Little Florets — Website Bundle
================================

This zip contains BOTH versions of the site. Pick one:

OPTION A — Easiest (single file)
  Use: index_standalone.html
  Just rename it to index.html and upload it alone (plus the images/ folder).
  CSS is inlined, so no site/ folder is needed.

OPTION B — Clean multi-file structure
  Use: index.html + site/ + images/
  Upload all three to your GitHub repo, preserving folder structure.

All asset paths are RELATIVE, so the site works under any subpath
(e.g. username.github.io/your-repo/).
